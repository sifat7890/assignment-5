Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a structured representation of an HTML document,where each part (tags,text,etc) is treated as an object in a tree-like hierarchy.It allows Javascript t dynamically access,modify and manipulate the content,structure and styles of a webpage.

How do you select an element from the DOM?

To select an element from the DOM we use getElementById, getElementsByClassName, querySelector, or querySelectorAll.

What is event delegation in the context of the DOM, and why is the useful?

Event delegation is a technique where a single event listener is attached to a parent element, allowing it to handle events from child elements efficiently, even if they are added dynamically.

How do you manipulate an event's attributes and style using the DOM?

We can manipulate an events attributes using event.target.setAttribute() and styles with event.target.style to dynamically change the elements properties when an event occurs.
